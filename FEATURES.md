# 🌌 Street Network: Platform Features

A high-fidelity creative and administrative suite for professional role-play.

## 📸 1. Screenshot Editor (The Forge)
- **Non-Destructive Editing**: Real-time image processing.
- **Narrative Overlays**: Name Manager & Dialogue System.
- **Advanced Redaction**: Code-based (`//`) and Manual Masks (Blur/Pixelate).
- **Post-Processing Filters**: Vignette, Digital Noise, Contrast, etc.
- **Strip Builder**: Vertical collage creator with drag-and-drop.
- **Integrated Crop Editor**: Native canvas cropping.
- **Undo/Redo System**: Full history support.

## 🧬 2. The Nexus (Infinite Spatial Canvas)
- **Infinite Navigation**: Fluid pan and zoom.
- **60FPS Dragging**: Hardware-accelerated movement logic.
- **Interactive Sticky Notes**: Color-coded notes with persistent storage.
- **Red Strings**: Pulsing connection lines for relationship mapping.
- **Backend Sync**: Automatic PostgreSQL persistence with status indicators.

## 🛠️ 3. Message Builder
- **Discord Utility**: Visual embed builder and webhook management.
- **Template Library**: Save and reuse message formats.

## 🛡️ 4. Admin & Security
- **User Management**: Control roles and account status.
- **Audit Logs**: Filterable history of all system actions.
- **Secure Sessions**: HTTP-only cookies and CSRF protection.

## 💎 5. Aesthetics & UX
- **Cyber-Noir Design**: Premium dark theme with Framer Motion animations.
- **Optimization**: GPU-accelerated rendering and dynamic code loading.
